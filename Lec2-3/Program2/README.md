Программа чтения данных из файла.  

Выделяем память для массива чаров с помощью функции calloc. Затем создаем дескриптор и открываем файл для чтения, если файла нет, он будет создан. Выводим значение дескриптора файла. После читаем из файла 10 байт и записываем в переменную `sz` количество байт, которое удалось прочитать. Затем обязательно записываем в конец массива, в который записывали прочитанную информацию, символ конца строки (терминальный ноль). В конце программы закрываем файл.  

Тестирование 
```
fd = 3  
called read( 3, c, 10). returned that 0 bytes were read.  
closed the fd.  
```
